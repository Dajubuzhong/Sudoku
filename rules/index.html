Game Rules

How to Play Sudoku

  1. Each row must contain the numbers 1-9 (or 1-6 for easy mode) without repetition
  2. Each column must contain the numbers 1-9 (or 1-6 for easy mode) without repetition
  3. Each 3x3 box (or 2x3 box for easy mode) must contain the numbers without repetition
  4. Use logic and deduction to fill in the empty cells
  5. The puzzle is solved when all cells are filled correctly
